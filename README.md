# QR Code Component
Languages  
[English](#english) | [Français](#français) | [한국어](#한국어)

A simple QR code card component built with HTML and CSS.

---

## English

This is my completed solution for the **QR Code Component** challenge on Frontend Mentor.  
The project was built using only HTML and CSS, with a focus on layout alignment, spacing, and visual hierarchy.

### Tech Stack
- HTML5
- CSS3 (Flexbox)
- Box Model (margin, padding, box-sizing)

### Features
- Card centered both horizontally and vertically
- Clean card-style UI using border-radius
- Depth and emphasis created with box-shadow
- Flexbox-based layout structure

### What I Learned
- How to center elements vertically and horizontally using Flexbox
- The difference between `padding` and `margin` and when to use each
- How `box-sizing: border-box` simplifies layout calculations
- How images interact with `border-radius`
- Creating visual depth using `box-shadow`

Frontend Mentor의 **QR Code Component** 챌린지를 완성한 프로젝트입니다.

- box-shadow를 활용한 카드 UI 표현
